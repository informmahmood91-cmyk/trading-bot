# Trading Bot
Webhook bot: TradingView → Gemini + ChatGPT → Telegram

## Setup
1. Add these 4 environment variables on Render:
   - TELEGRAM_TOKEN
   - TELEGRAM_CHAT_ID
   - GEMINI_KEY
   - CHATGPT_KEY
   - FINNHUB_KEY
   - TWELVE_DATA_KEY

2. TradingView webhook URL:
   https://your-app.onrender.com/webhook
